# Imágenes de Hospedajes

Esta carpeta contiene las imágenes de todos los hoteles y hospedajes de Cómbita.

## Nomenclatura de archivos
Usar el siguiente formato para nombrar las imágenes:
- `nombre-hospedaje.jpg` (ej: `quinta-rosita.jpg`, `hotel-sol-norte.jpg`)

## Especificaciones técnicas
- Formato: JPG o PNG
- Tamaño recomendado: 800x600px
- Peso máximo: 500KB por imagen
- Calidad: Optimizada para web
