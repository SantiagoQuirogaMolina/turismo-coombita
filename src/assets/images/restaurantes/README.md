# Imágenes de Restaurantes

Esta carpeta contiene las imágenes de todos los restaurantes de Cómbita.

## Nomenclatura de archivos
Usar el siguiente formato para nombrar las imágenes:
- `nombre-restaurante.jpg` (ej: `el-mana.jpg`, `la-gran-via.jpg`)

## Especificaciones técnicas
- Formato: JPG o PNG
- Tamaño recomendado: 800x600px
- Peso máximo: 500KB por imagen
- Calidad: Optimizada para web
